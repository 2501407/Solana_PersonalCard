# Solana Proof-of-Personhood Protocol (SolPassport)
Getting Started
Prerequisites
    Node.js v16+ and npm/yarn
    Solana CLI tools
    A Solana wallet (Phantom, Solflare, etc.)

# Installation
# Clone the repository
git clone https://github.com/2501407/Solana_PersonalCard.git
cd Solana_Personhood

# Install dependencies
npm install

# Development
npm run dev
